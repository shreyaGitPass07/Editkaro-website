## About the Project

A complete multi-page website for **Editkaro.in**, a social media marketing and video editing agency. Built with HTML, CSS, and vanilla JavaScript — no frameworks.

## Pages

- **Home** — Hero section, animated stats, filterable video portfolio, email subscription form
- **About** — Agency story, mission & vision, team members
- **Contact** — Project brief form with full validation

## Key Features

- 27-video portfolio across 9 categories with animated filtering and lightbox player
- Email subscription form → saves to Google Sheets via Apps Script
- Contact form → saves to Google Sheets via Apps Script
- Fully responsive (mobile, tablet, desktop)
- SEO optimised — meta tags, Open Graph, Schema.org on all pages

## Tech Stack

`HTML5` `CSS3` `JavaScript (ES6+)` `Google Apps Script` `Google Sheets`
